# Roadmap

## Phase 0: Foundation

- Next.js App Router application shell
- TypeScript strict mode
- Tailwind CSS
- Prisma and PostgreSQL
- Vitest and Playwright
- CI quality gates
- Content-safety documentation

## Phase 1: Authentication and Learner Model

- Replaceable authentication provider integration
- User profile and onboarding persistence
- Initial study-plan domain services
- Audit logging

## Phase 2: Reviewed Content Pipeline

- Reviewer workflow
- Approved content metadata
- Server-side release controls
- Demo fixture labeling
- Administrative import preview, coverage reporting, and report moderation

## Phase 3: Practice and Audio

- Question-attempt flow
- Audio queue
- Post-submission explanations
- Adaptive review scheduling

## Phase 4: Essays and Analytics

- Essay submission flow
- Post-submission sample-answer access
- Mastery analytics
- Admin reporting
