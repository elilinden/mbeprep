# Product

MBE Prep is an audio-first adaptive bar-preparation application for structured study, practice, review, essays, analytics, and planning.

## Current Scope

This repository currently contains the production foundation: routing, application shell, UI primitives, authentication, authorization, onboarding, database connectivity, tests, and documentation. It does not include legal instruction, MBE questions, essay prompts, answer explanations, or scoring rules.

## Product Principles

- Audio-first flows should support eyes-free review and short study sessions.
- Adaptivity should be explainable to learners and grounded in reviewed content.
- Legal content must be traceable, approved, and never improvised by software.
- Learners should never receive correct answers, explanations, or essay sample answers before submitting.
- Learners should confirm their resolved exam track before planning features use it.
